# Pick a Flower — 日历小花

一个极小的 web app：主界面是当前月历，用户点击今天的日期会随机生成一朵小花并添加到该用户的日历界面。每个用户名对应独立的数据，保存在浏览器的 localStorage 中。

快速开始

1. 安装依赖并启动（需要 Node.js）：

```bash
cd /Users/nora/Documents/Nora/app/pick_a_flower
npm install
npm start
```

2. 在浏览器打开：http://localhost:3000

使用说明

- 首次打开时输入一个用户名（任意文本），每个用户名拥有独立的日历数据（本地保存）。
- 在日历中点击任意日期会打开小花预览，点击“保存到选中日期”可将花保存到该日期的日历。特别地，点击“今天”的日期会立即生成并保存一朵随机小花（这是你的需求中的行为）。

开发说明

- 前端：静态文件位于 `static/`，主逻辑在 `static/app.js`。
- 后端：非常简单的静态文件服务器 `server.js`（仅用于本地开发）。

后续可选增强

- 使用真实后端与账户系统（便于跨设备同步）。
- 为每个保存的花添加元数据（时间、标题、颜色主题等）。
- 支持删除/编辑已保存的花。
