cast_memory 冥想盆 — 记忆封存容器

cast_memory 是一个受《哈利·波特》冥想盆启发的交互式记忆容器。

使用：
用户触碰银色液体后，柔和的魔法 UI 浮现，引导输入一段记忆与专属密钥。
点击 Cast Memory 后，系统使用算法对记忆进行加密，并生成一个可分享的短链接。任何持有链接与密钥的人，都可以解锁并重新召回这段记忆。
解锁时，液体中心逐渐变亮，旋涡加速，淡银色的 Memory Threads（记忆丝线） 从液面升起，最终解密出的文字以漂浮的魔法文本形式呈现。

Visual & Interaction Direction
	•	整体氛围：安静、古老、魔法、极简、电影感
	•	液体：分层旋涡、雾状银白、柔光，无金属感、无硬边
	•	记忆丝线：细、轻、丝绸般发光，缓慢上升
	•	粒子：极少量漂浮微尘
	•	光效：Bloom 与光晕极度克制，仅作为情绪增强

Pensieve — An Encrypted Memory Vessel

Pensieve is an interactive memory experience inspired by the Harry Potter Pensieve.

How to use：
When a user touches the luminous silver liquid, a soft magical interface emerges, inviting them to enter a memory and a private key.
After casting the memory, the system encrypts the text using AES algorithm and generates a short, shareable link. Anyone with both the link and the key can unlock the memory.
During decryption, the liquid brightens at its center, the vortex accelerates, and delicate silver Memory Threads rise from the surface. The decrypted text then appears as floating, magical typography within the scene.

Visual & Interaction Direction
	•	Mood: quiet, ancient, magical, minimal, cinematic
	•	Liquid: layered vortices, misty silver-white glow, soft light, no metallic edges
	•	Memory Threads: thin, weightless, silk-like strands of light rising slowly
	•	Particles: extremely subtle floating dust
	•	Lighting: restrained bloom and halos to suggest gentle magic
